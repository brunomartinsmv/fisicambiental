# data/raw

**PT**
Arquivos originais, sem modificacao. Exemplos: planilhas baixadas do INMET e arquivos de origem com falhas ou valores ausentes. Estes arquivos nao devem ser alterados; qualquer tratamento deve gerar novos arquivos em `data/cleaned/`.

**EN**
Original files with no modification. Examples include INMET spreadsheets and raw sources with missing values. Do not edit these files; any treatment should produce new files in `data/cleaned/`.

## Como reproduzir / How to reproduce
- Baixar os dados originais (INMET ou outras fontes)
- Salvar aqui mantendo o nome e formato original
